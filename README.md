## Installation
#### Prerequisites
- Tor or i2p installed and configured
- Packages: `build-essential`, `libsdl2-dev`, `libsdl2-ttf-dev`, `libsdl2-image-dev`, `libcurl4-openssl-dev`

```bash
git clone https://github.com/arhhhhh404/Lain-Requester.git
cd Lain-Requester
make
./lain
```
